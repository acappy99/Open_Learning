# Command prompt reference

### Create new directory
```
mkdir [folder_name]
```

### Create new file
```
touch [script.py]
```

### Shutdown, Sleep, Reboot
In order of title
```
sudo shutdown -h now

sudo shutdown -s

sudo shutdown -r
```
## Shutdown in so many minutes with a final warning message
```
sudo shutdown -h +15 "Computer will shutdown now"
```
